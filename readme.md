### Manual Installation

* **Open :** raw [bcs.js](https://raw.githubusercontent.com/AnimeAbout/BiliBili-Custom-Subtitles/main/bcs.js) and **Copy** all content in bcs.js
* **Open :** Tampermonkey Extension
* **Goto :** Dashboard
* **Click : +** Button and **Paste** replace all content
* **Save : Ctrl + s**
